# Simon

<figure><img src="../../../../.gitbook/assets/image (2).png" alt=""><figcaption></figcaption></figure>

Inspired by the Simon colors game that is actually a memory game, Nitrocid now has a dedicated game for simulating this game. Simon is a game that chooses random sets of four colors for you to memorize and repeat. If you got one color wrong, the game is over.

***

## <mark style="color:$primary;">Game Controls</mark>

Below controls are supported:

<table><thead><tr><th width="130">Keybinding</th><th>Description</th></tr></thead><tbody><tr><td><code>1</code></td><td>Selects green color</td></tr><tr><td><code>2</code></td><td>Selects red color</td></tr><tr><td><code>3</code></td><td>Selects blue color</td></tr><tr><td><code>4</code></td><td>Selects yellow color</td></tr><tr><td><code>ESC</code></td><td>Exits the game</td></tr></tbody></table>
