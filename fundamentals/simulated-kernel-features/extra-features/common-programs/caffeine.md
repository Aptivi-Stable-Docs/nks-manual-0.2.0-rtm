# Caffeine

<figure><img src="../../../../.gitbook/assets/image (3) (1) (1) (1).png" alt=""><figcaption></figcaption></figure>

If you need to be notified whether your coffee is ready after a specified time, you'll need to use this program to facilitate this task. You can use the caffeine command to provide it with either the number of seconds until your caffeine is ready or the caffeine name, which will be provided with the below table:

***

## <mark style="color:$primary;">Available types</mark>

You can refer to the list of available types of caffeinfe below.

| Caffeine name   | Time to prepare (seconds) |
| --------------- | ------------------------- |
| American Coffee | 300                       |
| Red Tea         | 600                       |
| Green Tea       | 600                       |

***

## <mark style="color:$primary;">Methods of usage</mark>

You can use the following command variants:

<table><thead><tr><th width="190">Method</th><th>Description</th></tr></thead><tbody><tr><td><code>caffeine &#x3C;seconds></code></td><td>Specifies the number of seconds to notify you</td></tr><tr><td><code>caffeine &#x3C;name></code></td><td>Specifies the caffeine name above</td></tr><tr><td><code>caffeine -abort</code></td><td>Aborts the pending caffeine notification</td></tr></tbody></table>
