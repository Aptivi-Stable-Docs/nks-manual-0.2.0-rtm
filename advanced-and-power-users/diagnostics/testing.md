# Testing

<figure><img src="../../.gitbook/assets/image (21).png" alt=""><figcaption></figcaption></figure>

Another way to diagnose the kernel is through testing. It uses the facade system to allow you to test individual kernel features. It can be invoked by running Nitrocid KS with the `testInteractive` command-line argument.

***

## <mark style="color:$primary;">Controls</mark>

In the main screen, it shows you all the available test facades, with the four below options. Pressing <kbd>ENTER</kbd> in the second pane will run a test facade and gives you an option to see whether the test succeeded or failed at the end of the test.
